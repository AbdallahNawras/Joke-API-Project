		JokeAPI Documentation

Overview
In this project, I am building a website using the Express/Node.js platform, with the Axios HTTP client, that integrates a chosen public API from the given list.The website should interact with the chosen API, retrieve data, and present it in a user-friendly manner.

Learning Objectives

Develop an understanding of how to integrate public APIs into web projects.
Gain practical experience using Express/Node.js for server-side programming.
Enhance understanding of client-server communication using Axios.
Demonstrate ability to manipulate, present, and work with data retrieved from APIs.

For the public API, I choose the JokeAPI to Create a website that supports a variety of filters that can be applied to get just the right jokes you need.

Link:https://v2.jokeapi.dev/
